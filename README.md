# BookVerse-Library-Management-System
A python-based library management system that allows users to register books, borrow and return books, manage library records, and view statistics through a menu=driven interface.
## ✨ Features

- Register new books
- View the complete library catalog
- Search books by title
- Borrow books
- Return books
- View borrowed books
- View available books
- Remove books from the library
- Display library statistics
- Automatically generate book codes (BK-001, BK-002, ...)

---

## 🛠 Technologies Used

- Python 3
- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements

---

## 📖 Book Information Structure

Each book contains:

- Book Code
- Title
- Author
- Category
- Status
- Borrower

Example:

```python
{
    "code": "BK-001",
    "title": "Atomic Habits",
    "author": "James Clear",
    "category": "Self Development",
    "status": "Available",
    "borrower": ""
}
```

---

## 📋 Menu Options

```text
1. Register New Book
2. View Library Catalog
3. Find Book By Title
4. Borrow Book
5. Return Book
6. View Borrowed Books
7. View Available Books
8. Remove Book
9. Library Statistics
10. Exit
```

---

## 📊 Library Statistics

The system can display:

- Total Books
- Available Books
- Borrowed Books

Example:

```text
Total Books: 10
Available Books: 7
Borrowed Books: 3
```
